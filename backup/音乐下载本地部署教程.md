# 音乐下载本地部署教程
此方法最好要有网易云的会员账户 不然只能下载不需要vip的歌
> [!TIP]
>急用的话可以先用[这个](https://ydl.baka.plus/)

来源是 [网易云音乐无损解析](https://github.com/Suxiaoqinx/Netease_url)
其仓库主页也有本地部署教程 并且有[在线的网页](https://wyapi.toubiec.cn/) 但是偶尔会失效 所以最好还是本地部署 以便随时可用

> [!IMPORTANT]
> 本地部署不可用的时候记得回到github的仓库页面查看最新修改发布的markdown文件

## step1 准备工作
1.需要为你的电脑准备git环境以方便克隆项目 点此[下载](https://registry.npmmirror.com/-/binary/git-for-windows/v2.52.0.windows.1/Git-2.52.0-64-bit.exe) git

2.为你的电脑安装python 安装
安装python有两种方法 
​法一：微软商店安装 点开你电脑的微软商店 直接搜索python下载即可

法二：官网安装 点此进入python[下载页](https://www.python.org/downloads/windows/) 

尽量​选择最新的stable版（稳定版）下载 

​安装完后win+r 输入powershell回车进入命令行窗口 复制粘贴下面代码检查python和git是否安装成功

检查python：
```
python --version
```
检查git：
```
git --version
```

显示出版本号即为成功 如下图所示

<img width="495" height="97" alt="Image" src="https://github.com/user-attachments/assets/29973ced-407a-469e-a646-f8777b015448" />

## step2 克隆项目
在你想部署的位置新建一个文件夹 比如桌面上 把文件夹名字改成英文 比如说wangyiyun然后点进文件夹 点击图示的那个框 

<img width="1117" height="631" alt="Image" src="https://github.com/user-attachments/assets/5fc0515a-be9b-4b7b-a023-edb908386a85" />

可以输入东西 然后我们输入cmd后回车 就会出现命令行窗口 如下图所示 

<img width="1483" height="762" alt="Image" src="https://github.com/user-attachments/assets/2c77e0e2-7379-4c6f-967c-0cb948638f40" />

接着复制下面这串代码 
```
git clone https://github.com/Suxiaoqinx/Netease_url.git
cd Netease_url
```
直接粘贴进去 然后电脑就会自己开始克隆项目到本地了

## step3 安装依赖 
等待克隆完成 可以再次输入命令后 复制粘贴以下代码
```
pip install -r requirements.txt
```
会自动开始安装依赖 

## step4 填写cookie
安装完成后退出命令行 可以看到文件夹里有个cookie.txt的文件

<img width="1408" height="821" alt="Image" src="https://github.com/user-attachments/assets/1b459dda-beab-451a-9a3b-557ccaa1bc2f" />

> [!IMPORTANT]
> 我们要在这个文件里填写我们的cookie 以此来调用网易云的api下载音乐

1.点此跳转[网易云音乐官网](https://music.163.com/) 登录自己的账号
2.点此下载[cookie编辑器](https://microsoftedge.microsoft.com/addons/detail/cookieeditor/neaplmfkghagebokkhpjpoebhdledlfi) （用来抓取cookie）安装完拓展后回到网易云官网 点开拓展 会出现这个页面

<img width="985" height="634" alt="Image" src="https://github.com/user-attachments/assets/378b040b-c61b-462f-ab52-b4bc1db2c8f8" />

点绿色圈内的music_u 把value里的内容复制下来

> [!TIP]
> 要全部复制 不要有漏 ctrl+a全选后复制

复制以下代码
```
MUSIC_U=“你刚复制的cookie”
```
这时打开cookie.txt文件
把你刚复制的cookie替换掉“你刚复制的cookie”后粘贴进去 如图

<img width="615" height="539" alt="Image" src="https://github.com/user-attachments/assets/ee98486d-dc18-4517-8c7a-e747ca19b5f6" />

ctrl+s保存后退出 cookie部分完成

## step5 启动服务
在Netease_url文件夹中

<img width="1408" height="821" alt="Image" src="https://github.com/user-attachments/assets/4d1f462c-e74f-4115-9434-9d84aca1fdfa" />

同样在顶部对话框内输入cmd回车进入命令行窗口

复制粘贴以下代码
```
py main.py
```
回车运行 出现下面这个页面即为成功

<img width="1483" height="762" alt="Image" src="https://github.com/user-attachments/assets/0adc9e35-9d52-4adc-aa6c-2b914d262e66" />

这时摁住ctrl鼠标左键点击http://127.0.0.1:5000/ （鼠标移到网址上会有下划线）就会自动跳转到浏览器 打开这个页面

<img width="1521" height="853" alt="Image" src="https://github.com/user-attachments/assets/1808aa06-ed90-4c72-91b1-6f1a30ba810c" />

即可搜索歌曲进行下载 

## step6 关闭服务

在命令行窗口摁ctrl+c即可退出
