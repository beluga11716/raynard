# 音乐下载本地部署教程

> [!TIP]
>急用的话可以先用[这个](https://ydl.baka.plus/)

来源是 [网易云音乐无损解析](https://github.com/Suxiaoqinx/Netease_url)
其仓库主页也有本地部署教程 并且有[在线的网页](https://wyapi.toubiec.cn/) 但是偶尔会失效 所以最好还是本地部署 以便随时可用

> [!IMPORTANT]
> 本地部署不可用的时候记得回到github的仓库页面查看最新修改发布的markdown文件

## step1 准备工作
1.需要为你的电脑准备git环境以方便克隆项目 点此[下载](https://registry.npmmirror.com/-/binary/git-for-windows/v2.52.0.windows.1/Git-2.52.0-64-bit.exe) 
2.为你的电脑安装python 安装












## step2 克隆项目
在你想部署的位置新建一个文件夹 比如桌面上 把文件夹名字改成英文 比如说wangyiyun然后点进文件夹 点击图示的那个框 

<img width="1117" height="631" alt="Image" src="https://github.com/user-attachments/assets/5fc0515a-be9b-4b7b-a023-edb908386a85" />

可以输入东西 然后我们输入cmd后回车 就会出现命令行窗口 如下图所示 

<img width="1483" height="762" alt="Image" src="https://github.com/user-attachments/assets/2c77e0e2-7379-4c6f-967c-0cb948638f40" />

接着复制下面这串代码 
```
git clone https://github.com/Suxiaoqinx/Netease_url.git
cd Netease_url
```
直接粘贴进去 然后就会自己开始克隆项目到本地了
